# etoro-app

SvelteKit app for eToro portfolio management.

## Runtime and Package Manager

**Use Bun** instead of Node.js, npm, pnpm, yarn, or vite.

- `bun install` instead of `npm install` / `pnpm install` / `yarn install`
- `bun run <script>` instead of `npm run` / `pnpm run` / `yarn run`
- `bunx <package>` instead of `npx`
- `bun <file>` instead of `node <file>` or `ts-node <file>`
- `bun test` instead of `jest` or `vitest`
- `bun build` instead of webpack, esbuild, or vite
